# Portafolio de QA Testing

Este repositorio muestra un proyecto práctico de **testing manual y testing de APIs** como parte de mi formación como QA Tester.

El objetivo es demostrar habilidades básicas de aseguramiento de calidad (QA), incluyendo diseño de casos de prueba, reporte de bugs, documentación de evidencias y pruebas de APIs.

---

# Descripción del proyecto

En este proyecto se realizaron pruebas sobre:

- Una aplicación web de demostración
- Una API pública utilizada para practicar testing

Las actividades realizadas incluyen:

- Diseño de **casos de prueba**
- **Ejecución de pruebas manuales**
- **Reporte de bugs**
- Documentación de **evidencias**
- **Testing de API** con validaciones automáticas

# Testing de API

Las pruebas de API fueron realizadas utilizando **Postman**.

Se probaron endpoints de una API pública de prueba.

Validaciones realizadas:

- Código de estado HTTP
- Tiempo de respuesta
- Estructura del JSON
- Validación de campos obligatorios

Colección de Postman:
postman/API Testing.postman_collection.json
